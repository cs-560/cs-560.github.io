Description

This is a demo simulating GUI for a VR app.

- Move with the arrow keys or WASD. (disabled when a menu is open)

- Look around with the mouse.

- Right Mouse Button opens menu in front of player.

- Use Left Mouse Button with center of view to interact with UI components.

Notes

Feb 14th. 2017 : The current build does not implement scene/clip switching or playback functionality.
